# MailForge AI Landing Page

MailForge AI is a modern, single-page SaaS landing page for an AI email writing assistant. The UI is dark-first, minimal, and motion-led, with glassmorphism cards, smooth section transitions, subtle parallax, and responsive layouts.

## Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- Framer Motion
- Lucide React icons
- Shadcn/UI
- pnpm

## Highlights

- Structured sections: Hero, Problem, How It Works, Features, Demo, Use Cases, Testimonials, Pricing, Final CTA, Footer
- Reusable design system (tokens, shared components, animation presets)
- Minimal line icon system with centralized icon mapping
- Scroll-triggered animations and layered parallax effects
- Dark/light theme support
